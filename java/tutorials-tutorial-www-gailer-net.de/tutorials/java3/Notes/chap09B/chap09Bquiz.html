Quiz über Ausdrücke

Das ist ein Übungsquiz. Die Ergebnisse werden nirgendwo erfasst und haben keine Auswirkung auf Ihre Note. Die Fragen in diesem Quiz dürfen in keinem Quiz oder Test erscheinen, der für Ihre Benotung zählt.

Hinweise: Entscheiden Sie sich bei jeder Frage für die beste Antwort. Klicken Sie dann auf das entsprechende Optionskästchen, um diese Antwort auszuwählen. Sie können Ihre Antwort jederzeit ändern. Sobald das Quiz bewertet wird, erscheint die richtige Antwort in der Textbox unterhalb der Frage.


1. Welcher der folgenden ist kein korrekter arithmetischer Ausdruck?

a. alpha
b. (alpha / impuls) - 12.4
c. alpha ( / impuls - 12.4 )
d. ((alpha / impuls) - 12.4)

2. Was ist ein Operand?

a. Ein anderer Name für eine Zahl.
b. Ein anderer Name für einen Integer.
c. Einer der arithmetischen Symbole wie *, +, - und /.
d. Ein Wert, auf den ein Operator einwirkt.

3. Folgende Deklarationen vorausgesetzt:

short  a = 12 ;

float  x = 2.3 ;
double y = 3.14 ;

Welche der folgenden Ausdrücke wird nicht als Gleitpunkt-Operation ausgewertet?

a. a + x
b. 1.5 + 8.6
c. a + b
d. x / y

  }

b. Kosten pro Artikel: 0
