Geben Sie das erste Wort ein:
turtle
Geben Sie das zweite Wort ein:
153

turtle.....................153
Das könnte für ein Inhaltsverzeichnis verwendet werden.

Hinweis: Verwenden Sie System.out.print(".") innerhalb einer Schleife, um die Punkte auszugeben.
