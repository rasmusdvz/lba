Füllen Sie die Lücken aus

Hinweise:   Das ist eine unbewertete Lückenübung. Jede Frage besteht aus einem Satz, bei dem ein oder zwei Wörter fehlen. Jede Schaltfläche repräsentiert das fehlende Wort bzw. die fehlenden Wörter. Überlegen Sie welches Wort bzw. welche Wörter fehlen. Klicken Sie dann auf die Schaltfläche, um zu sehen ob Sie recht haben. Es wird keine Note für diese Übung ermittelt.

1.Wählen Sie true oder false für jedes Paar der Optionskästchen. Klicken Sie dann auf den "?"-Button, um das Ergebnis der AND-Verknüpfung zu sehen.

true
false 		  AND   true
false 		    ergibt    

2.Wählen Sie true oder false für jedes Paar der Optionskästchen. Klicken Sie dann auf den "?"-Button, um das Ergebnis der OR-Verknüpfung zu sehen.

true
false 		  OR   true
false 		    ergibt    

3.  Für die AND-Verknüpfung wird in Java der -Operator verwendet.

4.  Für die OR-Verknüpfung wird in Java der -Operator verwendet.

5.  Arithmetische Operatoren (wie *, +, -, /) haben eine Präzedenz als die folgenden Operatoren (wie < , >, >=, &&, || ), so dass Arithmetik ausgeführt wird.

6.   Relationale Operatoren (wie <, >, <=) haben eine Präzedenz als logische Operatoren (wie &&, || ), so dass der Vergleich von Zahlen als ausgeführt wird. Da die logischen Operatoren die geringste Präzedenz haben, werden sie ausgeführt.

7.   1+2 > 4-2 && 12 < 23  ergibt  .

8.   1+2 > 4-2 || 12 < 23  ergibt  .

9.   1+2 > 4-2 && 12 > 23  ergibt  .

10. 1+2 > 4-2 || 12 > 23  ergibt  .

Ende des Tests. Wenn Sie ihn wiederholen möchten, laden Sie die Seite in Ihrem Browser neu.
