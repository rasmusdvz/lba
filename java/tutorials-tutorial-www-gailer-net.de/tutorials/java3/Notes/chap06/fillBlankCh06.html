Füllen Sie die Lücken aus

Hinweise:   Das ist ein unbewerteter Lückentest. Jede Frage besteht aus einem Satz, bei dem ein oder zwei Wörter fehlen. Jede Schaltfläche repräsentiert das fehlende Wort bzw. die fehlenden Wörter. Überlegen Sie welches Wort bzw. welche Wörter fehlen. Klicken Sie dann auf die Schaltfläche, um zu sehen ob Sie recht haben. Es wird keine Note für diese Übung ermittelt.

1. Eine ist eine von einem Programmierer mit einem Texteditor in der Sprache Java erstellte Textdatei.
    Eine -Datei ist eine vom Java Compiler erzeugte Datei in Maschinensprache für die Java Virtuelle Maschine.

2. Wählen Sie in dem folgenden Programm die richtigen geschweiften Klammern, linke geschweifte Klammer { oder rechte geschweifte Klammer } aus. 
  class Bye

     public static void main ( String[] args )
   
      System.out.println("Hello");
3. Die   Methode ist die Stelle an der ein Programm gestartet wird.
4. Ein   ist ein "grammatikalischer Fehler" bei der Verwendung einer Programmiersprache.
5. Während der Programmerstellung müssen Sie häufig Ihre Quelldatei überarbeiten, den Bytecode kompilieren und das Programm ausführen, um das Ergebnis zu sehen. Dieser Vorgang wird - -und- Zyklus genannt.
6. Angenommen, dass sich Ihr Programm ohne Probleme kompilieren läßt, aber beim Ausführen etwas nicht so ist, wie es sein sollte. Das wird ein genannt.
7. Ihre Aufgabe ist es ein Programm zu erzeugen, das die folgenden Zeilen auf den Bildschirm ausgibt: 
Klabautermann
Klabauterfrau
Klabauterkind
im Schiffe sind.
Vervollständigen Sie das Programm so, dass es seine Aufgabe erfüllt. 
class Klabautermann
{
  public static void main ( String[] args )
  {
    System.out.println("  ");
    System.out.println("  ");
    System.out.println("  ");
    System.out.println("  ");
  }
}
8. Eine ist ein ausführbares Java Bytecode-Programm, das keinen Webbrowser benötigt.
9. Hier ist ein schlecht eingerücktes Programm. Schlagen Sie eine bessere Möglichkeit das Programm einzurücken vor, indem Sie auf die Schaltflächen klicken, wo ein Einrücken angebracht ist. 
class Klabautermann
{
public static void main ( String[] args )
{
System.out.println("Klabautermann");
System.out.println("Klabauterfrau");
System.out.println("Klabauterkind");
System.out.println("im Schiffe sind.");
}
}

Hinweis: Funktioniert nur bei neueren Browsern (z.B. Opera 7.x, Firefox, Internet Explorer 6).
