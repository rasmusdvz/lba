xn = x * x * x * ... * x
     --------------------
           n Mal
Der Anwenderdialog sieht ungefähr so aus: 
Geben Sie x ein:


1.3
Geben Sie n ein:
5



1.3 hoch 5 ergibt: 3.7129300000000014

-------

Geben Sie x ein:
5.6
Geben Sie n ein:
-3
