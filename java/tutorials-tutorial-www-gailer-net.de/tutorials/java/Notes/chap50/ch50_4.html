Antwort:

Ja, eine Elternklasse kann eine beliebige Anzahl von Kindern haben. Aber eine Kindklasse kann nur eine Elternklasse haben.

Austauschbare Ausdrücke

Es gibt drei Begriffspaare, um das Verhältnis der Vererbung zu beschreiben: Eltern/Kind, Superklasse/Subklasse und Basisklasse/abgeleitete Klasse. Programmierer verwenden alle drei gleichermaßen. Die Abbildung zeigt die äquivalenten Paare.
