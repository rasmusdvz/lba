Konsistente Anwendung einer Regel

Der Betrag liegt derzeit bei $1000. Wenn Sie Schwierigkeiten hatten sich daran zu erinnern, hätten Sie auch jedes Mal dann Probleme das Programm korrekt zu programmieren, wenn ein Scheck verarbeitet werden müsste. Glücklicherweise wurde diese "Regel" schon in die Zugriffsmethode verarbeiteAuszahlung() programmiert. Dadurch, dass bei der Handhabung von Schecks immer diese Methode verwendet wird, ist die korrekte Anwendung dieser Regel immer gewährleistet: 

    bobsKonto.verarbeiteAuszahlung( 1400 );

Wenn es einen Fehler in der Anwendung der Regel gibt, gibt es nur eine Stelle, um nach dem Problem zu sehen: in der Zugriffsmethode.
