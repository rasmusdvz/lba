Kapitel 17 — Beispiele von Zählschleifen

Die Beispielprogramme in diesem Kapitel zeigen den typischen Gebrauch von Zählschleifen.

Beispiele in diesem Kapitel:

  • Addieren von geraden und ungeraden Ganzzahlen.
  • Fakultät N berechnen.
  • Einen Ziegelstein von einem Turm werfen.
  • Ausgabe eines Rechtecks von Sternchen.

Mit diesen Beispielen üben Sie das Erstellen von Programmen aus elementaren Teilen.

FRAGE 1:

Würden Sie gerne alle geraden Ganzzahlen von 0 bis Tausend addieren?
