Überblick über die Klasse Object

Diese Art der Programmierung ist sehr gebräuchlich. Sie werden das in Zukunft noch häufig sehen. Beachten Sie, dass keine neuen Objekte erzeugt wurden; sondern verschiedene Referenzvariablen für bereits existierende Objekte verwendet wurden.

In Java, gibt es eine vordefinierte Klasse, die der ultimative Vorfahre aller anderen Klassen ist. Diese Klasse wird Object genannt. Wenn Sie eine eigene Klasse so wie diese definieren: 

class MeineNeueKlasse
{

}

definieren Sie tatsächlich eine Subklasse der Klasse Object. Das folgende ist mit dem oben stehenden äquivalent: 

class MeineNeueKlasse extends Object
{

}

Jede Klasse in Java — Ihre eigenen und jede Klasse in jeder Bibliothek stammt von der Klasse Object ab. Wenn Sie zu grafischen Benutzeroberflächen kommen, wird es wichtig sein sich daran zu erinnern.
