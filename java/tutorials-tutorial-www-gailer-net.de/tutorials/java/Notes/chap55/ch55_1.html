CHAPTER 55 — Einführung in GUI-Programmierung

Dieses Kapitel führt in die Programmierung grafischer Benutzeroberflächen (GUI) ein. Anwender kommunizieren mit modernen Applikationsprogrammen mit Hilfe grafischer Komponenten wie Fenstern, Befehlsschaltflächen, Eingabefeldern und Menüs. Es wäre schwierig eine GUI-Applikation von Grund auf zu schreiben. Glücklicherweise wurde schon das meiste der Arbeit für Sie getan, in einer Gruppe von Klassen mit dem Namen Swing.

Themen:

  • Ereignisgesteuerte Programmierung
  • Grafische Benutzeroberflächen
  • Die drei Teile eines GUI
  • GUI-Komponenten
  • Container
  • Das Abstract Windowing Toolkit (AWT)
  • Wie das AWT mit Swing zusammenhängt
  • Swing Klassenhierarchie

Swing ist ein großes Paket. Diese Aufzeichnungen besprechen nur die wichtigsten Grundlagen. Weitere Informationen finden Sie in der Dokumentation von Oracle.
