Kapitel 9A — Variablen und Zuweisungsanweisungen

In allen, mit Ausnahme der kleinsten Programme, wird ein laufendes Programm ständig mit Werten arbeiten. Diese Werte werden in kleinen Abschnitten des Hauptspeichers aufbewahrt und Variablen genannt.

Themen:

  • Variablen
  • Zuweisungsanweisungen
  • Ausdrücke
  • Arithmetische Operatoren
