Syntax der Zuweisungsanweisung

Das Programm gibt das gleiche aus, wie das erste Programm. Aber, dieses Programm initialisiert die Variable nicht und muss also später einen Wert in sie hineinstellen.

Die Syntax von Zuweisungsanweisungen sieht so aus: 

variablenname  =  ausdruck ;
  • Das Gleichheitszeichen "=" bedeutet "Zuweisung".
  • variablenname ist der Name einer Variablen, die irgendwo in dem Programm deklariert wurde.
  • ausdruck ist eine Kollektion von Zeichen, die einen Wert ergeben.

Hier sind ein paar beispielhafte Zuweisungsanweisungen (setzen Sie voraus, dass die Variablen bereits deklariert wurden): 

gesamt = 3 + 5;

preis = 34.56;

steuer = gesamt*0.05;

In der Quelldatei muss eine Variable deklariert sein, bevor irgendeine Zuweisungsanweisung sie verwenden kann.
