Ausdruck 16 - 12 / 4 2 + 6 / 2 8 + 4*2 8+4 * 2 12/2 - 3 6/8 + 2
Wert 13 5 16 16 3 2

Das letzte Ergebnis ist korrekt. Zuerst wird 6/8 ausgewertet. Dabei wird Ganzzahldivision verwendet. Das Ergebnis ist 0. Dann wird diese 0 zur 2 addiert.

Auswertung Schritt für Schritt

Beim Auswerten eines Ausdrucks kann es hilfreich sein Schritt für Schritt vorzugehen und den Ausdruck nach jedem Schritt noch einmal zu schreiben. Zum Beispiel: 

16 - 12 / 4

Führen Sie zuerst die Division durch, da sie die höchste Präzedenz hat. Als nächstes schreiben Sie den Ausdruck neu, indem Sie die Division durch ihren Wert ersetzen: 

16 - 3

Werten Sie jetzt den neu geschriebenen Ausdruck aus:

13

Sie können diesen Vorgang auch so schreiben: 

16 - 12 / 4
     ------
16 -    3
---------
   13

Die gestrichelten Linien zeigen, was bei jedem Schritt getan wird.
