Syntax einer Variablendeklaration

Das Wort Syntax bedeutet die Grammatik einer Computersprache. Wir können über die Syntax eines kleinen Teils des Programms sprechen, wie über die Syntax einer Variablendeklaration.

Es gibt verschiedene Möglichkeiten der Variablendeklaration: 

datentyp   variablenname;
  • Dies deklariert eine Variable, ihren Datentyp und reserviert Speicher für sie. Die Deklarartion sagt nichts über den Wert aus, der in den Speicher gestellt wird. (Sie werden später in diesen Aufzeichnungen lernen, dass unter bestimmten Umständen die Variable automatisch initialisiert wird und dass sie unter anderen Umständen nicht initialisiert wird.) 
datentyp   variablenname  =  initialwert ;
  • Dies deklariert eine Variable, deklariert ihren Datentyp, reserviert Speicher für sie und stellt einen Anfangswert in diesen Speicher. Der Anfangswert muss vom richtigen Datentyp sein.
datentyp   variablennameEins, variablennameZwei ;
  • Dies deklariert zwei Variablen, beide vom gleichen Datentyp, reserviert Speicher für beide, aber stellt nichts in die jeweiligen Variablen. Wenn Sie wollen, können Sie das mit mehr als zwei Variablen tun. 
datentyp   variablennameEins  =  initialwertEins,
           variablennameZwei  =  initialwertZwei ;
  • Dies deklariert zwei Variablen, beide von dem gleichen Datentyp, reserviert Speicher und stellt einen Anfangswert in jede der Variablen. Sie können das alles in einer Zeile schreiben, wenn genug Platz vorhanden ist. Noch einmal, Sie können das für mehr als zwei Variablen tun, solange Sie dem Schema folgen.
