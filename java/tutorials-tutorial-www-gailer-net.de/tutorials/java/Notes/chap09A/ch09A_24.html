Werten Sie bei Gleichheit der Präzedenz von links nach rechts aus

Wenn es zwei (oder mehr) Operatoren gleicher Präzedenz gibt, wird der Ausdruck von links nach rechts ausgewertet. Beispiele: 

2 * 7 * 3
-----
  14  * 3
  -------
     42

4 - 2 + 5
-----
  2   + 5
  -------
      7

Gewöhnlich macht es nichts, ob die Auswertung von links nach rechts oder in einer anderen Reihenfolge durchgeführt wird. In der Algebra macht es keinen Unterschied. Aber in der Gleitpunktmathematik macht es manchmal einen wichtigen Unterschied.
