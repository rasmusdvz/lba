Quiz über Variable und Zuweisungsausdrücke


Das ist ein Übungsquiz. Die Ergebnisse werden nirgendwo erfasst und haben keine Auswirkung auf Ihre Note. Die Fragen in diesem Quiz dürfen in keinem Quiz oder Test erscheinen, der für Ihre Benotung zählt.

Hinweise: Entscheiden Sie sich bei jeder Frage für die beste Antwort. Klicken Sie dann auf das entsprechende Optionskästchen, um diese Antwort auszuwählen. Sie können Ihre Antwort jederzeit ändern. Sobald das Quiz bewertet wird, erscheint der Buchstabe der richtigen Antwort in der Textbox unterhalb der Frage.


1. Welches der folgenden ist ein reserviertes Wort?

a. apple
b. 7up
c. grandTotal
d. boolean

2. Welcher der folgenden ist KEIN korrekter Variablenname?

a. 2schlecht
b. zero
c. derLetzteWertVonAllen
d. jahr2000

3. Welche der folgenden Deklarationen ist NICHT korrekt?

a. double pflicht;
b. float kredit;
c. boolean wert = 12;
d. int start = 34, ende = 99;

4. Was gibt das folgende Programm aus? 

class Frage4
{
  public static void main ( String[] args )
  {
    int stueckkosten = 8;
    int anzahl       = 5;
    System.out.println("Gesamtkosten: " + (stueckkosten * anzahl) );
  }
}

a. Gesamtkosten: + 40
b. Gesamtkosten: 8*5
c. Gesamtkosten: 40
d. "Gesamtkosten: " 40

5. Welche der folgenden zeigt die Syntax einer Zuweisungsanweisung?

a. variablenname  =  ausdruck;
b. ausdruck  =  ausdruck;
c. ausdruck  =  variablenname;
d. datentyp  =  variablenname;

6. Welche zwei Schritte finden statt, wenn eine Zuweisungsanweisung ausgeführt wird?

a. (i) Der Ausdruck wird ausgewertet und (ii) das Ergebnis wird in der Variablen gespeichert.
b. (i) Der Wert wird in der Variablen gespeichert und (ii) der Ausdruck wird ausgewertet.
c. (i) Es wird Speicherplatz reserviert und (ii) die Zahl wird hineingeschrieben.
d. (i) Die Variable wird ausgewertet und (ii) das Ergebnis wird gespeichert.

7. Was ist ein Ausdruck?

a. Das gleiche wie eine Anweisung.
b. Ein Ausdruck ist eine Liste von Anweisungen, aus denen ein Programm besteht.
c. Ein Ausdruck ist eine Kombination von Literalen, Operatoren, Variablen und Klammern, um einen Wert zu berechnen.
d. Ein Ausdruck ist eine Zahl, die in Ziffern ausgedrückt wird.

8. Welcher der folgenden Ausdrücke ist NICHT korrekt?

a. 5 + 4
b. (5 * 4)/2
c. (-12 + (13 + 7)/2 ) * 4
d. -12 + (13 + 7)/2 ) * 4

9. Welcher der folgenden Ausdrücke ist falsch?

a. (34 - 86) / 3
b. (34 - 86) / -3
c. 34 - 86) / (23 - 3 )
d. ( (34 - 86) / (23 + 3 ) )
