Wenn ein Fuhrpark-Objekt konstruiert wird, wie viele neue Objekte gibt es?

Antwort:

Drei — das Fuhrpark-Objekt selbst und die zwei Auto-Objekte, auf die es verweist.

Abbildung eines Fuhrpark-Objekts

Die Abbildung zeigt, dass Objekte der Klasse Fuhrpark zwei Referenzen auf Auto-Objekte enthalten.
