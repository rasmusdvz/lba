Kapitel 53B — Die Schnittstelle Comparable

Eine Klasse implementiert die Schnittstelle Comparable, wenn ihre Objekte in einer bestimmten Reihenfolge sortiert werden können. Zum Beispiel implementiert die Klasse String Comparable, da Strings in alphabetischer Reihenfolge sortiert werden können. Numerische Klassen (wie Integer oder Double) implementieren Comparable, da Zahlen in numerischer Ordnung sortiert werden können.

Objekte einer Klasse, die Comparable implementiert, können miteinander verglichen werden, um zu bestimmen, welches zuerst kommt, wenn Objekte sortiert werden.

Themen:

  • Die Schnittstelle Comparable
  • Die Methode comparable()
  • Natürliche Reihenfolge
  • Reihung
  • Sortieren
  • Binäre Suche

Dieses Kapitel beschreibt die Schnittstelle Comparable für Java 5.0 (und höher), das Generics verwendet. Ältere Versionen von Java funktionieren anders.
