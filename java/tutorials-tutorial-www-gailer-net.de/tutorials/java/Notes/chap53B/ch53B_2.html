Comparable<T> Schnittstelle

Eine Schnittstelle besteht aus Konstanten und Methodendeklarationen. Eine Klasse, die eine Schnittstelle implementiert muss alle Methoden, die in einer Schnittstelle aufgeführt werden, implementieren. Die Comparable<T> Schnittstelle besteht nur aus einer Methode (und keinen Konstanten): 

int compareTo( T obj )  // Vergleiche dieses Objekt mit obj, das vom Typ T ist.
                        // Gib einen negativen Integer, Null oder einen positiven Integer zurück,
                        // wenn dieses Objekt kleiner, gleich oder größer ist als obj.

Hier steht T für den Typ des Objekts. Wenn zum Beispiel die Objekte Strings sind, dann ist der Typ von T String.

Wenn ein Objekt von einer Klasse ist, die Comparable implementiert, dann ist dieses Objekt kleiner, gleich oder größer als jedes andere Objekt dieser Klasse. Die Methode compareTo() gibt einen Integer zurück, um zu zeigen, welche der drei Relationen zutrifft.

  Relation   objektA.compareTo( objektB )
objektA Kleiner als objektB Negativer Integer
objektA Gleich objektB 0
objektA Größer als objektB Positiver Integer

Zum Beispiel:

"Apfel".compareTo("Orange") Negativer Integer
"Apfel".compareTo("Pflaume") Negativer Integer
"Apfel".compareTo("Apfel") 0
"Orange".compareTo("Orange") 0
"Orange".compareTo("Apfel") Positiver Integer

Nur das Vorzeichen des zurückgegebenen Integers zählt, wenn der Rückgabewert nicht 0 ist. Die Größe des zurückgegebenen Integers hat keine Bedeutung.

FRAGE 2:

Prüfen Sie die folgenden Deklarationen. Sie verwenden die Wrapperklasse Integer, die am Ende des Kapitels 9C besprochen wurde. Ein Integer-Objekt hält einen Integer als seine Daten und besitzt mehrere nützliche Methoden, so wie compareTo), um mit Integern zu arbeiten. 

Integer minusZehn = new Integer( -10 );
Integer minusFuenf = new Integer( -5 );
Integer fuenf = new Integer( 5 );
Integer zehn = new Integer( 10 );
Integer fuenfzehn = new Integer( 15 );

Was ist das Ergebnis der folgenden Vergleiche?

fuenf.compareTo( zehn )
zehn.compareTo( fuenf )
fuenf.compareTo( fuenf )
zehn.compareTo( fuenfzehn )
minusFuenf.compareTo( zehn )
minusFuenf.compareTo( minusZehn )

Hinweis: Bei zahlen ersetzen Sie gedanklich "compareTo" durch Subtraktion und führen die Arithmetik durch. Zeim Beispiel, fuenf.compareTo( zehn ) funktioniert wie 5 - 10.
