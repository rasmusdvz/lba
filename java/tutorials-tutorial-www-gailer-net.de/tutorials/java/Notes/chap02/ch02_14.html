Symbole und Muster

Erinnern Sie sich an diesen Vorteil des Binären:

  • Alles was mit Mustern dargestellt werden kann, kann mit Bitmustern dargestellt werden.

Was ist mit anderen Dingen als den geschriebenen Zeichen einer Sprache? Das ist ein sehr komplexes Thema und es wurden mehrere Bücher darüber geschrieben. Hier ist eine stark verkürzte Erörterung: Wählen Sie sich ein Thema aus. Beschreiben Sie es, indem Sie Sätze verwenden. Stellen Sie diese Sätze in ASCII (Zeichen kodiert als Bitmuster) dar. Jetzt ist das Thema binär dargestellt. Wenn etwas mit einer Sprache dargestellt werden kann, dann kann es auch binär dargestellt werden.

Beachten Sie, dass dies nichts über "Bedeutung" oder "Verständnis" aussagt. Gedruckte Bücher verstehen nicht selbst ihren Inhalt. Eine digitale Version des Buchs z.B. auf CD ROM wird von der CD ROM ebenfalls nicht verstanden. Sie enthält bloß die Information und wartet auf einen menschlichen Verstand, der sie versteht. Aber, das Buch wurde mit Bitmustern dargestellt.

Niemand hat gesagt, dass binäre Darstellungen leicht zu verwenden sind. Manche Methoden der Darstellungen sind für Computer sehr nützlich (zum Beispiel die Verwendung binärer Muster für Zahlen), andere Methoden sind fast nutzlos. Ein Hauptteil der Arbeit von Computerwissenschaftlern besteht darin sinnvolle Möglichkeiten herauszufinden, wie Dinge binär dargestellt werden können. Zum Beispiel wurden in den letzten zehn Jahren viel Arbeit hineingesteckt, um herauszubekommen wie sich Bild- und Audiodaten am besten darstellen lassen.

Wichtiger Punkt:

Alles was Computerspeicher (und andere Teile) enthalten sind Bitmuster. Was diese Bitmuster darstellen hängt von ihrer Verwendung ab.
