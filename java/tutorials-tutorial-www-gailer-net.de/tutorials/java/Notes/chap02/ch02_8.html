Binäre Signale

Analoge Signale ändern fortlaufend ihren Wert. Durch die Verwendung eines Grenzwertes können analoge Signale binäre Daten ("an/aus" Daten) darstellen. Es ist leicht und schnell (für Elektronik, als auch für Menschen) zu bestimmen ob eine Spannung sich oberhalb oder unterhalb eines Grenzwertes befindet. Die Abbildung zeigt ein Signal, das "aus" und dann "an" sendet. (Das Signal wird zu den Zeitpunkten T1 und T2 untersucht.)
