1. Was bedeutet das Wort binär?

a.    Binär bedeutet "enthalten in einem Computer".
b.    Binär bedeutet "nur zwei Zustände".
c.    Binär bedeutet "es gibt nur eine diskrete Anzahl von Werten".
d.    Binär bedeutet "Elektronik wird für Berechnungen verwendet".

2. Was ist ein Bit?

a.    Ein Bit ist ein einzelner binärer Wert.
b.    Ein Bit ist ein einzelnes im Hauptspeicher gespeichertes Zeichen.
c.    Ein Bit ist eine Sammlung von mehreren Bytes.
d.    Ein Bit ist eine kleine Einheit der Computerzeit.

3. Welche der folgenden Aussagen ist KEIN Vorteil der Konstruktion von Computern aus binären Geräten?

a.    Binäre Geräte sind einfach und leicht zu bauen.
b.    Binäre Signale sind eindeutig.
c.    Binäre Geräte sind viel schneller als dezimale Geräte.
d.    Bitmuster können verwendet werden um alles, was symbolisch ist, zu repräsentieren.

4. Welche der folgenden Aussagen ist wahr bei einem analogen Signal?

a.    Ein analoges Signal hat eine diskrete Anzahl von Zuständen.
b.    Ein analoges Signal ist die einzige Möglichkeit Musik aufzuzeichnen.
c.    Ein analoges Signal kann niemals in ein binäres Signal konvertiert werden.
d.    Ein analoges Signal ändert üblicherweise fortwährend seinen Wert.

5. Wenn ein analoges Signal Störgeräusche enthält, sind dann Informationen verloren gegangen?

a.    Ja — die Störungen überlagern die exakten Werte des ursprünglichen Signals.
b.    Nein — Elektronik kann die Störungen einfach ignorieren.
c.    Nein — dem Signal wurden zusätzliche Information hinzugefügt.
d.    Vielleicht — es hängt davon ab, wie stark die Störung ist.

6. Wenn ein binäres Signal Störgeräusche enthält, sind dann Informationen verloren gegangen?

a.    Ja — der exakte Wert der Bits kann nicht bestimmt werden.
b.    Nein — der exakte Wert der Bits kann bestimmt werden, solange die Störung nicht zu groß ist.
c.    Nein — Binäre Signale können nicht gestört werden.
d.    Ja — das Signal wird durch die Störung zusätzliche Bits haben.

7. Warum hat ein Computer einen Taktgeber?

a.    Der Zustand der binären Signale wird nur zu bestimmten Zeitpunkten gemessen.
b.    Ein Taktgeber wird benötigt, um zu überprüfen wie schnell sich die Signale ändern.
c.    Ein Taktgeber wird benötigt, um zu überprüfen ob das Spannungsniveau korrekt ist.
d.    Ein Taktgeber wird nur von Applikationsprogrammen verwendet, die die aktuelle Zeit brauchen.

8. Kann japanische Schrift vom Computer dargestellt werden?

a.    Nein — nur Englisch kann dargestellt werden.
b.    Nein—nur Sprachen mit einem Alphabet können dargestellt werden.
c.    Ja—aber es wird ein spezieller Prozessorchip benötigt.
d.    Ja—da sie symbolisch ist und alles was symbolisch ist dargestellt werden kann.

9. Kann englische Schrift durch analoge Signale dargestellt werden?

a.    Nein — nur binäre Signale können symbolische Daten darstellen.
b.    Nein — sie ist nicht symbolisch.
c.    Nein — analoge Signale können gar nichts darstellen.
d.    Ja — sprechen Sie einfach laut in ein Mikrofon.

10. Warum ist es wichtig, dass unbegrenzt, fehlerfreie Kopien von binären Daten gemacht werden können?

a.    Da Daten viele Male pro Sekunde in einem Computersystem hin und her kopiert werden.
b.    Das Übermitteln von Daten über das Internet führt dazu, dass viele Kopien von diesen Daten gemacht werden.
c.    Applikationsprogramme wie Textverarbeitungen und Computerspiele müssen perfekte Kopien der ursprünglichen sein, damit sie funktionieren.
d.    Alle drei oben stehenden Antworten sind richtig.
