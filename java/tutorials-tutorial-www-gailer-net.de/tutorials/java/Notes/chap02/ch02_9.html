Unvollständige Übertragung

Die "an" und "aus" des vorigen Signals sind deutlich. Aber was wenn das Signal über eine lange Leitung gesendet wird und jemand einen Staubsauger einschaltet? Die Kurve zeigt das Signal am anderen Ende der Leitung.

Obwohl das Signal verzerrt ist (auf der analogen Ebene), werden die binären Werte perfekt übertragen. Sie (und die Elektronik) können immer noch sagen, dass das Signal zum Zeitpunkt T1 ein "aus" darstellt und, dass es zum Zeitpunkt T2 ein "an" darstellt. Das Empfangsteil muss nur die binären Werte bekommen.


Da nur die "an" und "aus" Information zählt, ist das analoge Rauschen irrelevant und das ursprüngliche Signal wird perfekt empfangen (soweit es die binäre Information betrifft.)
