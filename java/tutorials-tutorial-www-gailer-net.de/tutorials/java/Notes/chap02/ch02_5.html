Vorteile des Binären 2: Eindeutige Signale

Das Schnitzen einer Buchseite in ein Holzkarree erfordert große technische und künstlerische Fähigkeiten. Die Aufgabe ist komplex und arbeitsintensiv. Ein kleiner Fehler könnte das gesamte Karree verderben. Die Holzschnitzer der Renaissance gehörten zu den fähigsten Holzschnitzern die es je gab.

Das Setzen einer Drucktype ist vergleichsweise leicht. Es bedarf nur geringer künstlerischer und technischer Fertigkeiten. Die einzelnen Drucktypen werden einfach an ihrer Stelle abgesetzt. Die Methode ist robust und verlässlich; Fehler können korrigiert werden. Die Drucktype kann viele Mal wieder verwendet werden.

Es ist paradox, dass ein Technologiefortschritt oft zu einfacheren Methoden führt. Holzschnitzen ist komplex; das Setzen von Drucktypen ist einfach. Die Computerrevolution (ca. 1950 — ) ist oft mit der Gutenberg Revolution (ca. 1450 — ) verglichen worden.

Betrachten wir noch einmal die Vorteile des Binären:

  1. Einfach; leicht herzustellen.
  2. Eindeutige Signale (daher störsicher).
  3. Fehlerfreie Kopien können gemacht werden.
  4. Alles, was in einer bestimmten Art von Muster dargestellt werden kann, kann mit Bitmustern dargestellt werden.
