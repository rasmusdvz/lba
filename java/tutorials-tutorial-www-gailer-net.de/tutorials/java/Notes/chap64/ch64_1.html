Kapitel 64 — Sliders und Change-Events

Dieses Kapitel bespricht die Klasse JSlider. Ein Slider ist eine Komponente, die die Bedienung eines Schiebereglers nachahmt, wie er manchmal bei Audiogeräten gefunden wird.

Sliders sind eine gute Wahl, wenn der Anwender einen Wert anpassen und dabei dessen Auswirkung beobachtet möchte. Das ist besser als einen bestimmten Wert in eine Textbox einzugeben.
