Kapitel 25 — Objekte

Bis jetzt haben wir noch nicht allzu viel mit Objekten zu tun gehabt. Dieses Kapitel behandelt Objekte und Klassen und, wie sich eine objektorientierte Sprache wie Java von anderen Sprachen unterscheidet.

Themen:

  • Was ist ein Objekt?
  • Was ist eine Klasse?
  • Charakteristische Merkmale von Objekten
  • Klassenmethoden (static)
  • Konstruktoren
  • Plätzchen-Backformen
  • Punktnotation

Vergessen Sie für eine Weile das Programmieren. Denken Sie an die Dinge in der Welt, die Objekte sind und an Dinge, die keine Objekte sind. Das ist ein wirklich tiefgreifendes Problem, das Philosophen mehrere tausend Jahre beschäftigt hat. Machen Sie sich keine allzu großen Sorgen, wenn Ihnen dieses Problem nicht klar ist.
