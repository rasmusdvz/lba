Wenn Sie 372 Pennies in Dollar eintauschen, wie viele Dollarscheine bekommen Sie? Wie viele Pennies bleiben übrig?

Antwort:

  • Die Anzahl der Dollarscheine ist 372 / 100 == 3
  • Übriggebliebene Pennies sind 372 % 100 == 72

Übung mit dem Modulo-Operator

Für positive Zahlen bedeutet   Ganzzahl % X   so viele X wie möglich in Ganzzahl zu packen, dann ist der übriggebliebene Rest der Wert des Ausdrucks. Versuchen Sie das Folgende:
