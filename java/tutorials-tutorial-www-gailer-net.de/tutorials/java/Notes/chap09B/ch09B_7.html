Ganzzahl und Gleitpunkt gemischt

Ganzzahldivison ist für sich betrachtet einfach zu sehen, aber hier ist ein mehr verwirrender Fall der gleichen Art: 

1.5 + 7/2

Die Division wird zuerst durchgeführt, da / eine höhere Präzedenz als + hat. Das Ergebnis 3 ist eine Ganzzahl. Jetzt wird die Gleitpunktzahl 1.5 zur Ganzzahl 3 addiert, um die Gleitpunktzahl 4.5 zu bekommen.

Ganzzahlarithmetik kann in Teilen eines Ausdrucks und in anderen nicht verwendet werden, auch wenn der letztendliche Wert des Ausdrucks ein Gleitpunkt ist.
