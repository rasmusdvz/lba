Der Aufruf des Konstruktors der Superklasse muss die erste Anweisung im Konstruktor von MeinFrame sein.

Container halten Referenzen auf Komponenten

MeinFrame ist ein Container, da er von JFrame erbt. Also können ihm GUI-Komponenten hinzugefügt werden. Wenn einem Container eine Komponente hinzugefügt wird, wird eine Referenz auf diese Komponente der Komponentenliste des Containers hinzugefügt.

Grafisch erscheinen die einem Container hinzugefügten Komponenten in dem Rechteck auf dem Bildschirm, das dem Container zugeteilt ist. In dem Beispielprogramm fügt der Code dem Frame ein JLabel hinzu und es erscheint innerhalb dieses Bereichs.
