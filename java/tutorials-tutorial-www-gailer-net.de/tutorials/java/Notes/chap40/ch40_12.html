Math.sin( 0.5 ) >= Math.tan( 0.2 ) || 43.259 / 12.073 > 3.5 || ( 100 > 45 )

Hier gibt es Seiteneffekte, so dass sicher ist nur den letzten Teil zu betrachten: 

( 100 > 45 )

und zu merken, dass der gesamte Ausdruck true sein ist. Java (in dem es von links nach rechts arbeitet) könnte ein anderes true gefunden haben, aber das spielt keine Rolle, da es keine Seiteneffekte gibt. Der gesamte Ausdruck ist true.

Sportwagen — Neu überdenken

Mathematische Funktionen wie Math.sin() und Math.cos() haben keine Seiteneffekte. Sie können problemlos in Boole'schen Ausdrücken verwendet werden.

Nehmen wir einmal an, Sie haben es noch nicht aufgegeben den $25.000 teuren roten Miata Sportwagen zu kaufen. Der Autoverkäufer macht Ihnen ein Angebot. Er verkauft Ihnen das Fahrzeug wenn

  • Sie haben mindestens $25,000 in bar ODER
  • Sie haben mindestens $25,000 Kredit UND keine außenstehenden Schulden.

Die Logik hinter dieser Situation ist ein wenig kompliziert. Hier sind einige Möglichkeiten:

  • Sie haben $30,000 in bar, keinen Kredit und massive Schulden.     Sie bekommen das Auto.
  • Sie haben $1,000 in bar, $50,000 Kredit und massive Schulden.     Sie bekommen das Auto NICHT.
  • Sie haben $1,000 in bar, $10,000 Kredit und keine Schulden.     Sie bekommen das Auto NICHT.
  • Sie haben $1,000 in bar, $30,000 Kredit und keine Schulden.     Sie bekommen das Auto.

Beachten Sie, dass es nicht ausreicht nur einen hohen Kreditrahmen zu haben. Um den Sportwagen auf Kredit kaufen zu können, müssen Sie auch schuldenfrei sein.
