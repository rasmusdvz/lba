Aber überprüfen Sie besonders die Klammern, wie "[" und "{" und "(". Sie müssen die richtigen verwenden. Beachten Sie auch die exakte Groß- und Kleinschreibung.

Tüfteln Sie ein Weile herum, um das Programm zum Laufen zu bringen. Wenn Sie es einfach nicht zum Laufen bringen, hören Sie auf und machen später weiter. Sie haben vermutlich ein triviales Detail übersehen, dass Sie dann klar erkennen werden. Das passiert die ganze Zeit beim Programmieren.
