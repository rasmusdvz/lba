Kapitel 5 — Einführung in Java

Themen:

  • Hello World Programm
  • Java Bytecode
  • Java Compiler und Java Virtuelle Maschine
  • Die Verwendung von Notepad (EDITOR), um ein Quellprogramm zu erzeugen
  • Ein Java Programm starten

Die vorherigen Kapitel haben Computerhardware, Computersoftware, Bits und Bytes, Maschinensprache, Compiler, Interpreter und virtuelle Maschinen behandelt. Dieses Kapitel setzt voraus, dass Sie das alles wissen und bereit sind Java zu betrachten. (Dieses Kapitel finden Sie im Tutorial Java 3.)

Diese Aufzeichnungen wurden für Java 2 Standard Edition Version 5.0 oder höher geschrieben. Dieser Ausgabe wurden viele Merkmale hinzugefügt und frühere Versionen werden mit diesen Aufzeichnungen nicht so gut funktionieren. Wenn Sie Java zum ersten Mal installieren, installieren Sie Java 2 Standard Edition Version 6.0 oder höher.
