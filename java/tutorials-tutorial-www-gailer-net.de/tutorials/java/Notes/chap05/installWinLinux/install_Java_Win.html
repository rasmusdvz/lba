Kleine Anleitung Java Installation unter Windows
AUSDRUCKEN

Java installieren:

1.) Grundsätzliches:

  • Lade dir einen Editor aus meiner Software Liste (unter "Erstis" > "EAD") herunter. Ich empfehle derzeit den JCreator, weil der realtiv leicht zu bedienen ist. Aber ein Editor ist immer eure Gewöhnungssache.


  • Außerdem brauchst du das JDK, deine Entwicklungsumgebung, ohne die Java nicht läuft. Die Exe Datei führt dich durch eine Setup Routine, wichtig ist hier nur, dass du dir den JDK Ordner (in den du installierst) merkst!


  • Nach der Installation des JDK und deines Editors mußt du die Pfadvariable für javac (den Java Compiler) setzen, um auch außerhalb des "bin" compilieren zu können. Dieser Vorgang unterscheidet sich je nach Betriebssystem etwas.

2a.) Installation unter Windows 95/98:

  • Normalerweise geht es folgendermaßen: Windows besitzt eine Datei namens autoexec.bat, welche sich mit einem Texteditor (z.B. Notepad) öffnen läßt.


  • Rechtsklick auf autoexec.bat, dann kannst du sicherheitshalber eine Kopie davon in einem Selbstangelegten backup Verzeichnis speichern, z.B. unter dem Namen autoexec.bk. (Diesen Schritt kannst du nach deinem eigenen Ermessen überspringen).


  • Nach dem Öffnen der autoexec.bat solltest du dort nach der Zeile PATH=C:\WINDOWS;[...] (die [...] steht für evtl. weitere Pfadregistrierungen auf deinem System) suchen. Dies ist die Pfadvariable deines Betriebssystems.


  • Dieser Pfadvariablen fügst du jetzt den Pfad zu deinem JDK bin Verzeichnis hinzu. Das ganze sollte dann von den anderen Pfaden mit einem Semikolon getrennt werden. Sieht dann in etwa so aus: PATH=C:\WINDOWS;C:\JDK1.3\bin;[...] (vorausgesetzt, dass C:\JDK1.3\bin das Verzeichnis ist, welches ihr für JDK Installation angelegt habt.)


  • So, jetzt alles speichern und schließen, am besten Windows neu starten und fertig seid ihr!

2b.) Installation unter Windows NT/2000:

  • Startmenü > Einstellungen > Systemsteuerung > System (Icon doppelklicken). Unter Windows NT den Karteireiter "Umgebung" auswählen, unter Windows 2000 den Reiter "Erweitert" wählen.


  • Dort jetzt unter Nutzer- und Systemvariablen nachschauen. Du suchst auch hier nach PATH. Falls du nicht sicher bist, welchen Pfad du nehmen sollst, dann füge unter "Nutzervariablen" am rechten Ende hinter einem Semikolon (falls andere Pfade bereits stehen) deine Pfadvariable hinzu C:\JDK1.3\bin (vorausgesetzt, dass C:\JDK1.3\bin das Verzeichnis ist, welches ihr für JDK Installation angelegt habt.) Es sollte desweiteren auch nur eine Angabe zu einem JDK in eurem PATH stehen.


  • Auf "OK" oder "Übernehmen" klicken.


  • Falls Kommandozeilenfenster geöffnet waren, diese schließen und erneut öffnen! Jetzt seid ihr fertig.

2c.) Installation unter Windows ME:

  • Startmenü > Programme > Zubehör > Systemprogramme > Systeminformationen (oder Startmenü > Ausführen und in die Zeile msinfo32 eintippen. Das Fenster, dass sich öffnet sollte den Titel "Microsoft Hilfe und Support" haben.


  • "Werkzeuge" Menü auswählen und dort das System Konfigurations Dienstprogramm starten.


  • Den Reiter "Umgebung" auswählen, dort PATH auswählen und auf "Ändern" klicken.


  • Dieser Pfadvariablen fügst du jetzt den Pfad zu deinem JDK bin Verzeichnis hinzu. Das ganze sollte dann von den anderen Pfaden mit einem Semikolon getrennt werden. Sieht dann in etwa so aus: C:\JDK1.3\bin;[...] (vorausgesetzt, dass C:\JDK1.3\bin das Verzeichnis ist, welches ihr für JDK Installation angelegt habt.)


  • Die Änderungen speichern und den Neustart bestätigen.

2d.) Installation unter Windows XP:

  • Startmenü > Einstellungen > Systemsteuerung > System (Icon doppelklicken). Den Reiter "Erweitert" wählen, dort links unten auf "Umgebungsvariablen" klicken.


  • path auswählen und auf "Bearbeiten" klicken.


  • Am rechten Ende hinter einem Semikolon (falls andere Pfade bereits gesetzt sind) deine Pfadvariable hinzufügen C:\JDK1.3\bin (vorausgesetzt, dass C:\JDK1.3\bin das Verzeichnis ist, welches ihr für JDK Installation angelegt habt.) Es sollte desweiteren auch nur eine Angabe zu einem JDK bin in eurem PATH stehen.


  • So, jetzt alles speichern und schließen, am besten Windows neu starten und fertig seid ihr!

3.) Probleme:

  • Manchmal gibt's den Fehler:
    Exception in thread "main" java.lang.NoClassDefFoundError: HalloWelt


  • Normalerweise geht es folgendermaßen: Windows besitzt eine Datei namens autoexec.bat, welche sich mit einem Texteditor (z.B. Notepad) öffnen läßt.


  • Evtl. liegt's an eurem Programm Aufruf (z.B. unter Unix), also, dass ihr java HelloWorld.class eintippt. Achtet darauf, dass dort keine Endung stehen darf, also: java HelloWorld


  • Rechtsklick auf autoexec.bat, dann kannst du sicherheitshalber eine Kopie davon in einem Selbstangelegten backup Verzeichnis speichern, z.B. unter dem Namen autoexec.bk. (Diesen Schritt kannst du nach deinem eigenen Ermessen überspringen).


  • Füge die Zeile set CLASSPATH=.;Pfad(in welchem jar und zip Dateien sind);Pfad(wo du deine Java Programme speicherst) ein


  • So, jetzt alles speichern und schließen, am besten Windows neu starten und fertig seid ihr!

Hoffentlich hilft euch das über die Anfangsschwierigkeiten mit Java hinweg!
