Software »» Installation Java unter Linux



Diese Anleitung erläutert Euch, wie Ihr mit wenigen Handgriffen Java auf Eurem Rechner installiert.

Folgendermaßen kannst du Java installieren:

1) am besten sind die Virtual Machines von Blackdown, da sie auf Linux optimiert ist und verschiedene Feature enthalten, die ansonsten nicht vorhanden sind. (z.B Verknüpfung für Java Webstart Anwendungen erstellen) Blackdown.org

2) kopiere die Datei in ein Verzeichnis, in dem du Schreibrechte hast

3) nachdem du über die Kommandozeile das Bin File aufgerufen hast, wurde das JDK in ein entsprechendes Unterverzeichnis entpackt

4) optional kannst du mit root Rechten dieses Verzeichnis in ein anderes z.B./usr/j2dk1.4.2 kopieren

5) als nächtes setzt du deinen PATH auf das Unterverzeichnis ../bin, also z.b. in dem unter 4) beschriebenen Pfad "export PATH=$PATH:/usr/j2sdk1.4.2/bin" (Ich habe dies in der Datei .bashrc in meinem Homeverzeichnis getan)

6) starte eine neue Konsole

7) gib ein java -version, dann sollte die entsprechende Version angezeigt werden
