Applikation mit einem Textfeld

Sie haben diese Dokumentation vielleicht schon auf Ihrem Computer. Wenn nicht, können Sie sie von Oracle herunterladen. Überprüfen Sie sie, um zu sehen, wie diese Aufzeichnungen mit der offiziellen Dokumentation übereinstimmen. Das Ziel dieser Aufzeichnungen ist es einen Überblick über das Swingpaket zu vermitteln. Betrachten Sie die Dokumentation als die vollständige Referenz.

Hier ist einer (von mehreren) Konstruktoren von JTextField. Der Parameter bestimmt die Breite des Textfeldes (in Zeichen). 

JTextField( int numChars )

Beachten Sie, dass der Parameter die Anzahl der Zeichen bestimmt und nicht Pixel oder eine andere Maßeinheit bedeutet. Dieses Programm zeigt ein Frame mit einem Textfeld an.
