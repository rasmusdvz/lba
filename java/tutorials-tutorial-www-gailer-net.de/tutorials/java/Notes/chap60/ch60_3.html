Die JTextField Klassenhierarchie

Das JTextField ist Teil des Swingpakets, also ist es in javax.swing definiert. Hier ist wie es in die Klassenhierarchie passt. Die Wolken in Lavendelfarbe sind Teil von Swing.

JTextField ist von javax.swing.JTextComponent abgeleitet. Es ist eine Komponente mit vielen Merkmalen, die fast die gesamte Funktionalität eines Texteditors unterstützt. Der Anwender kann den eingegebenen Text ändern, indem er die Rücktaste verwendet, er kann ihn löschen und die Pfeiltasten verwenden. Er kann den Text ziehen und ablegen, und mit der Zwischenablage arbeiten.
