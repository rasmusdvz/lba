Die Swingkomponete: JTextField

Ein JTextField ist eine Box, die eine Zeile Text enthält. Der Anwender kann den Text in die Box eingeben Das Programm kann den Text bekommen und ihn verwenden. Das Programm kann auch das Ergebnis einer Berechnung in ein JTextField schreiben. Hier ist ein Label und ein Textfeld (beide innerhalb eines Applets):

Um dieses Applet zu sehen, muss Ihr Browser Java aktiviert haben.

Um Text in das Feld eingeben zu können, müssen Sie zuerst mit der Maus hineinklicken. Das gibt dem Feld den Tastaturfokus (üblicherweise nur Focus genannt). Wenn eine Komponente den Fokus hat, werden Zeichen von der Tastatur zu dieser Komponente dirigiert. Eine Komponente kann den Fokus auf verschiedene Weise bekommen. Der Anwender kann sie anklicken oder das Programm kann ihr den Fokus geben.

Geben Sie etwas Text ein. Editieren Sie ihn mit der Rücktaste und löschen Sie. Alles das funktioniert. Aber das Feld ist mit keiner Applikation verbunden, so dass nichts mit dem Text getan wird.
