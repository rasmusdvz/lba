Kapitel 62 — JPanel und BoxLayout

Die grafischen Benutzerschnittstellen der vorherigen Kapitel hatten ein Problem: Wenn der Anwender die Größe des Frames ändert, ändert sich das Layout der Komponenten drastisch. FlowLayout hält zusammengehörende Elemente nicht zusammen. Es stellt sie in eine Zeile, eine nach der anderen, wie sie hinzugefügt werden. Ein Label und die dazugehörende Komponente können so auf zwei verschiedene Zeilen verteilt werden. Dieses Kapitel bespricht, wie das Layout besser kontrolliert werden kann.

Themen:

  • Die setResizable() Methode
  • Komponenten in einem JPanel anordnen
  • Verschachtelte JPanels
  • Der BoxLayout Layoutmanager

FRAGE 1:

(Rückblick:) Konnte der Anwender die Größe der GUIs der vorherigen Kapitel verändern?
