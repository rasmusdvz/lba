Kapitel 26 — Objektreferenzen

Das Kapitel behandelt zwei Dinge: Objektreferenzvariablen und die Objekte, auf die sie verweisen. Das ist ein Kernkonzept der Programmierung und kann verwirrend sein. Es erfordert ein sorgfältiges Studium. Dieses Kapitel wurde mit dem größtmöglichsten Bestreben geschrieben, Fragen von ins Schwimmen geratenen Studenten zu beantworten.

Themen:

  • Primitive Datentypen und Objekte
  • Objektreferenzen und Objektreferenzvariablen
  • Kopieren von Objektreferenzen
  • Der == -Operator und Objektreferenzen
  • Die equals() Methode von Strings
  • Stringliterale

Studieren Sie dieses Kapitel sorgfältig.
