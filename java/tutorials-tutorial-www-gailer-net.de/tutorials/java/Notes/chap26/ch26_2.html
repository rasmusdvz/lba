(Rückblick:) In Java ist ein Stück Daten entweder von einem primitiven Datentyp oder es ist ein Objekt-Datentyp.

Primitive Datentypen und Klassen

Java hat viele eingebaute Datentypen, und Sie (als Programmierer) können so viele definieren, wie Sie brauchen. Anders als die primitiven Datentypen, sind alle anderen Datentypen Klassen. Mit anderen Worten, Daten sind entweder primitive Daten oder Objektdaten. Den einzigen Datentyp, den ein Programmierer definieren kann ist ein Objektdatentyp (eine Klasse). Jedes Objekt in Java ist eine Instanz einer Klasse. Zuerst muss die Klassendefinition existieren, bevor ein Objekt konstruiert werden kann.
