Ein Pentium Prozessor

Das ist ein Pentium 90 Prozessor von meinem 1995 Dell Dimension Computer. Der tatsächliche Siliziumchip ist innerhalb des goldenen Metallrechtecks eingeschlossen. Die Abbildung zeigt den unteren Teil des Pentiumchips. Die vielen goldenen Stifte werden üblicherweise in einen Sockel der Hauptplatine des Computers gesteckt.
