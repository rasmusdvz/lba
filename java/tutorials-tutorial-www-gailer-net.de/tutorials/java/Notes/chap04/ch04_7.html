Unterschiedliche Prozessoren

Es gibt viele Prozessortypen, die in Computersystemen verwendet werden. Sie wissen vermutlich etwas über den bekanntesten von ihnen, den Intel Prozessor (486, Pentium, Pentium II) und die Prozessoren, die in Apple Computern verwendet werden. Ein Computersystem wird um seinen Prozessor herum entworfen. Die andere Elektronik wird für einen bestimmten Prozessortyp entworfen.

Der grundsätzliche Unterschied z.B. zwischen einem Apple Power Macintosh und einem Dell Corporation Computer ist der Prozessor. Unterschiedliche Prozessoren verfügen über unterschiedliche Maschinenoperationen. Ein Maschinenprogramm für einen Dell Computer (mit einem Pentium Prozessor) würde für einen Apple Computer keinen Sinn ergeben. Das ist der Grund, warum Sie gewöhnlich darauf achten müssen, dass Sie die korrekte Version einer Applikation für Ihren Computer zu Hause erwerben. Wenn Sie das Spiel MYST kaufen, brauchen Sie die Version für den Prozessor Ihres Homecomputers.

Aber es gibt eine Version von MYST sowohl für den Pentium als auch für den Apple Computer. Wie kann das gehen, wenn die Prozessoren unterschiedliche Maschinenoperationen ausführen? Die Antwort ist, dass selbst, wenn die Prozessoren unterschiedlich sind, die jeweils verfügbaren Maschinenoperationen verwendet werden können, um die gleichen Dinge zu tun. Beide Prozessortypen besitzen genug Leistung durch ihre kleinen Maschinenoperationen, so dass alles was der eine Prozessor mit dem einen Maschinenprogramm tun kann, der andere Prozessor mit dem anderen Maschinenprogramm tun kann.

Die Architektur eines Prozessors ist die Art ihrer Maschinenoperationen, wie sie organisiert und implementiert sind, und wie der Prozessor mit dem Hauptspeicher und anderen Komponenten kommuniziert. Architektur befasst sich mit dem allgemeinen Plan und den Funktionen eines Prozessors; sie hat nicht allzu viel mit den elektronischen Details zu tun. In den meisten Informatikfachbereichen wird ein Kurs über Computerarchitektur verlangt.
