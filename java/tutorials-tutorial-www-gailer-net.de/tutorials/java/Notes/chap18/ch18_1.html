Kapitel 18 — Überwachungsgesteuerte Schleifen

Die vorherigen Kapitel haben Schleifen besprochen und insbesondere Zählschleifen. Die gleichen Anweisungen, die verwendet werden um Zählschleifen zu erstellen, werden verwendet um andere Typen von Schleifen zu erstellen. Dieses Kapitel wird einen Schleifentyp betrachten, der als überwachungsgesteuerte Schleife bezeichnet wird.

Themen:

  • Überwachungsgesteuerte Schleifen.
  • Beispielprogramme: Listen von Ganzzahlen addieren und Polynome berechnen.
  • Die Verwendung einer if-else-Anweisung innerhalb des Schleifenkörpers.
  • Zusammengehörende if und else.
  • Ein String als Kontrollwert.

FRAGE 1:

Angenommen, dass ein Programm eine Liste von Zahlen addieren soll, die vom Anwender über die Tastatur eingegeben werden. Wird in diesem Programm eine Schleife verwendet werden?
