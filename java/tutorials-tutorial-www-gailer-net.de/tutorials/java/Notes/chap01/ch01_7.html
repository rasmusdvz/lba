Antwort:

  • Wo ist das Programm, mit dem Sie arbeiten?
    • Im Hauptspeicher.
    • Laufende Programme sind immer im Hauptspeicher.
    • Eine dauerhafte Kopie befindet sich im Sekundärspeicher auf der Festplatte.
  • Wo sind die Zeichen, die Sie getippt haben?
    • Im Hauptspeicher.
    • Wenn Sie Ihr Dokument "speichern", werden die Zeichen in eine Datei auf der Festplatte kopiert.

Sekundärspeicher

Programme und Daten werden langfristig im Sekundärspeicher aufbewahrt. Gebräuchliche Sekundärspeichergeräte sind Festplatte, CD, DVD und USB Stick.

  • Die Festplatte hat verglichen mit dem Hauptspeicher eine enorme Speicherkapazität.
  • Die Festplatte befindet sich üblicherweise in der Systemeinheit eines Computers.
  • Die Festplatte wird für die langfristige Speicherung von Programmen und Daten verwendet.
  • Daten und Programme auf der Festplatte werden in Dateienorganisiert.
    • Eine Datei ist ein Abschnitt auf der Festplatte, der einen Namen hat.

Eine Festplatte hat zur Zeit üblicherweise eine Speicherkapazität von 500 bis 1000 Gigabyte. Sie kann aber auch noch größer sein. (Dabei entsprechen 500 Gigabyte dem Platz von ungefähr 500 x 109 Zeichen. Das ist ungefähr das Hundertfache eines Hauptspeichers.) Eine Festplatte ist im Vergleich zum Hauptspeicher langsam. Wenn der Prozessor mit ihr direkt kommunizieren würde, würde sich das ganze Computersystem verlangsamen. Dieser Gegensatz ist der Grund, warum es zwei Speichertypen gibt:

PrimärspeicherSekundärspeicher
  1. Schnell
  2. Teuer
  3. Geringe Kapazität
  4. Arbeitet direkt mit dem Prozessor zusammen
  1. Langsam
  2. Preiswert
  3. Große Kapazität
  4. Nicht direkt mit dem Prozessor verbunden
