Hardware und Software

In dem Sie über Der Zauberer von Oz nachdenken, denken Sie über Informationen nach. Sie werden normalerweise weder über das tatsächliche Material nachdenken, das verwendet wird, um die Information zu speichern (Videofilm oder DVD), noch über das Gerät, das die Informationen verwendet (Filmprojektor oder DVD-Player). Eine DVD ist physisch, aber der Film selbst ist eine immaterielle, nicht fassbare Information. Wenn Sie über einen Film sprechen, meinen Sie gewöhnlich den immateriellen Film, nicht ein bestimmtes Gerät, das eine Aufnahme von ihm hat. (Auch wenn die menschliche Sprache bewundernswert flexibel ist: Wenn Sie jemand in einem Videogeschäft bittet ihm den "Zauberer von Oz" zu reichen, werden Sie ihm wahrscheinlich die richtige DVD geben.)

Das Wort Hardware wird für physische Geräte wie Fernseher, DVD-Player und Computer verwendet. Das Wort Software wird für die Information benützt, die mit solchen Geräten verwendet wird: Filme, Musik, Romane, Computerprogramme und Daten.

Wenn über Computersystem gesprochen wird, bedeutet Hardware die physischen Teile des Computers und Software die Programme und Daten, die von dem physischen Computer verwendet werden.
