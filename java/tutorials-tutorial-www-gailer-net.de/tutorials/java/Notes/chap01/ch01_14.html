Antwort:

Die Festplatte. Die Festplatte wird für die langfristige Speicherung der gesamten Software verwendet, inklusive des Betriebssystems.

Ein Programm starten

Wenn ein Computer gestartet wird, lädt die Hardware automatisch das Betriebssystem und startet es. Dieser Prozess wird booten genannt. Der Grund für diesen merkwürdigen Begriff ist, dass das Betriebssystem daran beteiligt ist sich zum Laufen zu bringen — ein Vorgang, der sich vergleichen läßt mit "sich selbst an den Stiefelriemen (Bootstraps) herauszuziehen". Sobald das Betriebssystem läuft, wird es verwendet, um andere Programme zu starten.

Hier ist eine (vereinfachte) Liste von dem was passiert, wenn der Anwender (Sie) eine Applikation startet. Setzen Sie voraus, dass das Betriebssystem (OS für Operating System) bereits läuft.

  1. Der Anwender verlangt den Start einer Applikation.
    • Das kann durch das Klicken auf ein Icon, einen Menübefehl oder durch etwas anderes geschehen.
  2. Das Betriebssystem stellt die Applikation fest.
  3. Das Betriebssystem findet den Abschnitt auf der Festplatte, auf dem das Applikationsprogramm und dessen Daten gespeichert sind.
  4. Das Betriebssystem findet einen freien Abschnitt des Hauptspeichers, der groß genug für die Applikation ist.
  5. Das Betriebssystem kopiert die Applikation und dessen Daten in diesen Abschnitt des Hauptspeichers.
    • Die Software auf der Festplatte bleibt unverändert; eine Kopie von ihr wurde im Hauptspeicher angelegt.
  6. Das Betriebssystem richtet Ressourcen für die Applikation ein.
  7. Schließlich startet das Betriebssystem die Applikation.

Während die Applikation läuft, managt das Betriebssystem im Hintergrund die Ressourcen, sorgt für den Input und Output der Applikation und hält alles am Laufen.
