Hauptspeicher.

Hauptspeicher

Im Hauptspeicher werden Programme und Daten gehalten, wenn der Prozessor sie aktiv verwendet. Wenn Programme und Daten aktiv werden, werden sie vom Sekundärspeicher in den Hauptspeicher kopiert, wo der Prozessor mit ihnen interagieren kann. Eine Kopie verbleibt im sekundären Speicher. Der Hauptspeicher ist sehr eng mit dem Prozessor verbunden, so dass der Transport der Programminstruktionen und der Daten in und aus dem Prozessor sehr schnell ist. Der Hauptspeicher wird manchmal als RAM bezeichnet. RAM steht für Random Access Memory. "Random" bedeutet, dass jede Speicherzelle in beliebiger Reihenfolge angesprochen werden kann.

Wenn Leute sagen, dass ein Computer "4 Gigabyte RAM" hat, sprechen Sie darüber wie groß sein Hauptspeicher ist. Ein Gigabyte entspricht 1024 Megabyte, und ein Megabyte Speicher ist genug, um ungefähr eine Million Zeichen einer Textdatei bereitzuhalten. (In diesen Aufzeichnungen wird später noch mehr auf Bytes, Mega- und Gigabytes eingegangen.) Nichts wird dauerhaft im Hauptspeicher aufbewahrt. Manchmal werden Daten nur für ein paar Sekunden in den Hauptspeicher gestellt; nur so lange wie sie benötigt werden.
