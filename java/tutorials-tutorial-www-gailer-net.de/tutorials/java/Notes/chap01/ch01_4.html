Hardwarekomponenten

Der Begriff Input und Output sagt aus, ob Daten in die Systemeinheit hinein oder herausfließen. Die Abbildung zeigt die wichtigsten Hardwarekomponenten eines Computersystems. Die Pfeile zeigen die Richtung des Datenflusses.


Der Bus ist eine Gruppe von Leitungen auf der Hauptplatine des Computers. Über ihn fließen die Daten zwischen den Komponenten. Die meisten Komponenten sind mit dem Bus durch einen Controller verbunden, der die Aktivitäten zwischen dem Bus und den Komponenten koordiniert.

Der Prozessor ist ein von Kunststoff umgebenes elektronisches Gerät von ungefähr einem Quadratzoll. Innerhalb des Quadrats befindet sich ein noch kleineres Quadrat aus Silizium, das Millionen von winzigen elektronischen Teilen enthält. Ein Prozessor kann 100 Millionen Transistoren enthalten. Der Prozessor ist das "Gehirn" des Computersystems. Er führt die elementaren Berechnungen des Systems aus und kontrolliert direkt oder indirekt alle anderen Komponenten.

Der Prozessor wird manchmal auch als Central Processing Unit oder CPU bezeichnet. Ein Computer wird einen bestimmten Prozessortyp, wie Pentium SPARC haben.
