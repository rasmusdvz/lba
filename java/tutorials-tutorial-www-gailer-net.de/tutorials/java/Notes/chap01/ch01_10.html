Eingabegerät. (Obwohl manche Controller taktiles Feedback geben und sowohl Ausgabe- als auch Eingabegerät sind.)

Eingebettete Systeme

Ein Computersystem, das Teil einer größeren Anlage ist und die Arbeitsweise dieser Anlage kontrolliert, ist ein eingebettetes System (Embedded System). Üblicherweise führt der Prozessor fortwährend ein einzelnes Steuerprogramm aus, das dauerhaft im ROM (Read Only Memory) hinterlegt ist.

Die überwältigende Mehrzahl der Prozessorchips wird in eingebetteten Systemen verwendet. Nur 0,2% der Prozessorchips wird in den bekannten Desktop-Computern verwendet.

Ein typisches eingebettetes System ist ein Mobiltelefon. Das ist offensichtlich, aber es gibt viele, weniger einleuchtendere eingebettete Systeme. Ihr Auto enthält dutzende von Prozessoren, ohne das Audiosystem mitzurechnen. Ein Airbag wird zum Beispiel durch seinen eigenen Computerchip kontrolliert.
