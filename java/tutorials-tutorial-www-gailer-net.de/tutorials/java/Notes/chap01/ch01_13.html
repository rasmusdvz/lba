Betriebssysteme

Ein Betriebssystem ist ein komplexes Programm, das die Hardware- und Softwarekomponenten eines Computersystems koordiniert und dafür sorgt, das sie funktionieren. Es ist mit einem Ladenbesitzer vergleichbar, der in seinem Laden alles in Ordnung hält (Lieferungen entgegennimmt, Regale auffüllt, die Buchhaltung macht usw.), während er auf Kunden wartet.

Die meisten Computer funktionieren potenziell mit verschiedenen Betriebssystemen. Die meisten Pentium-basierten Computer laufen sowohl mit Linux- als auch mit Windows-Betriebssystemen. Üblicherweise ist auf einem Computersystem nur ein Betriebssystem installiert, auch wenn manche Computer mehrere haben. Der Computernutzer wählt beim Starten des Computers welches Betriebssystem den Computer kontrolliert, bis er wieder abschaltet wird.
