Komplexere Ausdrücke

Der Ausdruck auf der rechten Seite des Gleichheitszeichens kann umfangreicher sein, so wie in dem folgenden Codefragment: 

int zahl = 10 ;
int ergebnis = 0 ;

ergebnis = zahl++ * 2 ;

System.out.println("Zahl: " + zahl + " Ergebnis: " + ergebnis );

Ich rate Ihnen dringend es zu vermeiden solche Ausdrücke zu schreiben. Aber andere Programmierer werden solche Ausdrücke schreiben, also müssen Sie in der Lage sein sie zu lesen. Sie könnten auf diese Ausdrücke treffen, wenn Sie anderen Programmierern helfen Ihre schlecht geschriebenen Programme zu debuggen.
