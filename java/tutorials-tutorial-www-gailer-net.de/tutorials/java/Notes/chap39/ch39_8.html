Dekrementoperator

Der -- Operator ist ein Postfix- und ein Präfix-Dekrementoperator. Der Postfix-Operator dekrementiert eine Variable nachdem ihr Wert verwendet wurde. Der Präfix-Operator dekrementiert eine Variable, bevor ihr Wert verwendet wird.

AusdruckOperationBeispielErgebnis
x++ verwende den Wert, dann addiere 1int x = 10;
int y;
y = x++ ;		 x ist 11; y ist 10
++x addiere 1, danach verwende den Wertint x = 10;
int y;
y = ++x ;		 x ist 11; y ist 11
x-- verwende den Wert, danach subtrahiere 1int x = 10;
int y;
y = x-- ;		 x ist 9; y ist 10
--x subtrahiere 1, danach verwende den Wertint x = 10;
int y;
y = --x ;		 x ist 9; y ist 9

Prüfen Sie den folgenden Code: 

int x = 99;
int y = 10;

y = --x ;

System.out.println("x: " + x + "  y: " + y );
