Präfix-Inkrementoperator

Der Inkrementoperator ++ kann vor eine Variable gesetzt werden. In diesem Fall ist er ein Präfix-Operator. Wenn der Operator hinter einer Variablen steht, ist er ein Postfix-Operator. In beiden Fällen wird die Variable inkrementiert. Jedoch:

++zaehler bedeutet Inkrementieren, bevor die Variable verwendet wird.
zaehler++ bedeutet Inkrementieren, nachdem die Variable verwendet wurde.

Wenn der Inkrementoperator als Teil eines arithmetischen Ausdrucks verwendet wird, müssen Sie zwischen Präfix- und Postfix-Operatoren unterscheiden. 

int summe = 0;
int zaehler = 10;

summe = ++zaehler;

System.out.println("Summe: "+ summe " + Zaehler: " + zaehler );

Dieses Codefragment erfordert eine sorgfältige Prüfung:

  • Der ++ Operator ist jetzt ein Präfix-Operator.
  • Jetzt wird die Variable zaehler inkrementiert bevor der Wert, den sie enthält, verwendet wird.
  • Die Zuweisungsanweisung wird in zwei Schritten ausgeführt:
    1. Auswerten des Ausdrucks auf der rechten Seite des Gleichheitszeichens:
      • Der Wert ist 11 (da zaehler inkrementiert wird, bevor die Variable verwendet wird.)
    2. Zuweisen des Wertes an die Variable auf der linken Seite des Gleichheitszeichens:
      • summe erhält den Wert 11.
  • Die folgende Anweisung gibt aus: summe: 11 zaehler: 11
