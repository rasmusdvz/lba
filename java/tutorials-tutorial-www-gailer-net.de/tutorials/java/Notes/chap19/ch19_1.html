Kapitel 19 — Ergebnisgesteuerte Schleifen

Ein dritter Typ von Schleifen kann aus den elementaren Kontrollanweisungen von Java erstellt werden. Das ist die ergebnisgesteuerte Schleife. (Andere Namen dafür sind freie Schleife und allgemeine Schleife). Eine ergebnisgesteuerte Schleife läuft bis die Berechnung ein bestimmtes Ziel erreicht hat. Es ist wie die Anweisung eines Plätzchenrezepts, das besagt "Rühren Sie solange bis alle Zutaten sorgfältig vermischt sind." Sie wissen erst nachdem Sie das gewünschte Ergebnis erreicht haben, dass Sie aufhören können.

Themen:

  • Ergebnisgesteuerte Schleifen.
  • Beispielprogramme: Sparkonto mit Zinsen und Newtons Methode.
  • Geschachtelte Schleifen
  • Eine Schätzung durch Iteration verbessern.
  • Genauigkeit von Variablen doppelter Genauigkeit.

FRAGE 1:

Sie haben der Bank $1000 gegeben, bei einem jährlichen Zinssatz von 5%. Wie viel werden sie am Ende des Jahres haben?
