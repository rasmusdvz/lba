int aktuellerKontostand()

Andere Namen für diese Methode wären selbstverständlich auch möglich.

Implementierung der Aktueller-Kontostand-Methode

Bis jetzt sieht unsere Klasse so aus: 

class Bankkonto
{
  // Instanzvariablen
  String kontonummer;
  String kontoinhaber;
  int    kontostand;

  // Konstruktoren
  . . . .

  // Methoden
  int aktuellerKontostand()
  {
     return  ;
  }

}

Der Rückgabetyp der Methode ist "int", da die Methode den aktuellen Kontostand zurückliefert, der ebenfalls ein int ist. Die Methode benötigt keinen Input, folglich ist die Parameterliste leer. Dennoch werden die beiden Klammern ( ) benötigt, auch wenn innerhalb dieser nichts steht.

Alles, was in aktuellerKontostand() eingetragen werden muss, ist lediglich eine Anweisung, die den Kontostand des Bankkontos zurückgibt.

Vervollständigen Sie die Methode durch das Ausfüllen der Lücke.
