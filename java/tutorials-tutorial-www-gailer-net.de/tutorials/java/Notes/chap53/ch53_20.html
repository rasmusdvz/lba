Zusätzliche Merkmale von Schnittstellen

Es gibt weitere Merkmale von Schnittstellen, die dieses Beispiel nicht zeigt: Eine Klasse kann mehrere Schnittstellen implementieren: 

class IrgendeineKlasse extends IrgendeineSuperklasse
    implements SchnittstelleA, SchnittstelleB, SchnittstelleC
{

}

Jetzt muss IrgendeineKlasse alle Methoden aller Schnittstellen implementieren.
