Kapitel 16 — Mehr über Zählschleifen

Dieses Kapitel vertieft Ihre Praxis mit Zählschleifen. Es gibt andere Schleifentypen als Zählschleifen, aber Zählschleifen sind bei weitem die gebräuchlichsten und einer weiteren Untersuchung wert.

Themen:

  • Um zwei hochzählen
  • Die while-Bedingung als Wächter
  • Iteration
  • Dekrementieren der Schleifenkontrollvariablen
  • Endlos-Schleifen
  • Einen Wert auf der Basis einer Schleifenkontrollvariablen berechnen
