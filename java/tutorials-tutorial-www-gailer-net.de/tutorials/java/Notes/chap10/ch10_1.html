Kapitel 10 — Input und Output

Dieses Kapitel erörtert Input und Output für Java 5.0 und höher. Die meisten Programme lesen Daten ein und geben die Ergebnisse aus. Dieses Kapitel bespricht Möglichkeiten Inputdaten über die Tastatur einzulesen und die Ergebnisse auf den Bildschirm auszugeben. Spätere Kapitel werden den Input und Output von Disk-Dateien und anderen Medien besprechen.

Themen

  • Ein- und Ausgabeströme
  • Der I/O-Standardstrom
  • Scanner-Klasse
  • Ausnahmen (Exceptions)
  • Character-Daten in int-Daten konvertieren

Hinweis: Manche Bücher verwenden vom Autor des Buchs geschriebene Klassen, um zu zeigen, wie Input geht. Um diese Klassen zu verwenden, müssen Sie ein Paket von CD oder einer Website installieren. Sie können diese Aufzeichnungen (mit den Java Standardmethoden) problemlos neben Ihrem Buch verwenden.
