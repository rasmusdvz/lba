Kapitel 20 — Die Klasse Random


Programme brauchen häufig Zufallszahlen. Ein Kartenspiel verwendet Zufallszahlen, um einen gemischten Kartenstapel zu erzeugen. Ein Würfelspiel verwendet Zufallszahlen, um einen Würfelwurf zu simulieren. Viele Aktivitäten in der realen Welt (wie Verkehr auf einem Autobahnnetz) haben den Anschein des Zufälligen und ein Programm, das diese modelliert, muss Zufallszahlen verwenden. Dieses Kapitel bespricht die Klasse Random, die ein Bestandteil von Java ist.

Zufallszahlen, Wahrscheinlichkeit und Statistik sind wichtig zum Testen, Modellieren der Welt und zum allgemeinen Verständnis des Universums. Informatik Studiengänge erfordern üblicherweise einen Kurs zu diesen Themen.
