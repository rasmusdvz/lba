Nur einer der Spieler wird gewinnen. Jede Runde addiert immer einen Punkt genau für einen Spieler, also kann nur ein Spieler die 10 Punkte zuerst erreichen.

  int spielerscore = 0, computerscore = 0;

  // Runden spielen, bis einer der Spieler den ENDSTAND erreicht hat
  while ( spielerscore < ENDSTAND && computerscore < ENDSTAND )
