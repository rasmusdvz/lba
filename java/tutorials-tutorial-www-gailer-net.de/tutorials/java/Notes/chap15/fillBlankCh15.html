Füllen Sie die Lücken aus

Hinweise:   Das ist eine unbewertete Lückenübung Jede Frage besteht aus einem Satz, bei dem ein oder zwei Wörter fehlen. Jede Schaltfläche repräsentiert das fehlende Wort bzw. die fehlenden Wörter. Überlegen Sie welches Wort bzw. welche Wörter fehlen. Klicken Sie dann auf die Schaltfläche, um zu sehen ob Sie recht haben. Es wird keine Note für diese Übung ermittelt.

Dieser Test wiederholt Schleifen und die while-Anweisung.

1.   Wenn eine Maschine eine bestimmte Tätigkeit immer wieder ausführt, spricht man von einem . Wenn ein Programm Anweisungen wiederholt, spricht man von einer .

2.  Füllen Sie die Lücken aus, um die Syntax der while-Anweisung zu zeigen: 
while   ( )
     

3.  Die Anweisung(en), die direkt auf die while-Anweisung folgt, wird genannt. Es kann sich dabei auch um eine   handeln, die mehrere Anweisungen innerhalb der geschweifter Klammern { } enthält.

4.   Was muss in der richtigen Reihenfolge beachtet werden, damit eine Schleife korrekt läuft?
  1. Der muss richtig gesetzt sein.
  2. Die in der while-Anweisung muss richtig sein.
  3. Die im Schleifenkörper muss richtig sein.

5.   Die Bedingung der while-Anweisung muss jedes Mal ergeben, bevor der Schleifenkörper ausgeführt wird.

6.   Die Variable (normalerweise eine int-Variable), die verwendet wird um eine Schleife zu kontrollieren, wird genannt.
