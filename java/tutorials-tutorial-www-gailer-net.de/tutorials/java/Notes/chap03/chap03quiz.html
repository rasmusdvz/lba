1. Was ist ein Bit?

a.    Ein einzelner An/Aus-Wert.
b.    Ein Wert von "eins".
c.    Ein großer Abschnitt Computerspeicher.
d.    Genug Speicher, um ein Zeichen zu speichern.

2. Welche der folgenden Aussagen ist FALSCH?

a.    Die Methode, die verwendet wird um ein Bit zu implementieren, hat keine Auswirkung auf die Information, die es darstellt.
b.    Ein Bit kann als ein winziger Teil einer Magnetplatte implementiert sein.
c.    Das Kopieren eines Bits von einem Medium auf ein anderes hat keine Auswirkung auf die Information, die es repräsentiert.
d.    Ein Bit ist immer als etwas elektronisches implementiert.

3. Wie wird eine Gruppe von 8 Bits üblicherweise genannt?

a.    Ein Bite.
b.    Ein Bight.
c.    Ein Byte.
d.    Ein Blight.

4. Wie viel Speicher ist ein Megabyte?

a.    Ungefähr 100 Bytes.
b.    Ungefähr 1000 Bytes.
c.    Ungefähr eine Million Bytes.
d.    Ungefähr eine Milliarde Bytes.

5. Wie viel ist 25 * 23 ?

a.    28
b.    210
c.    48
d.    106

6. Welche der folgenden Aussagen ist FALSCH?

a.    Jedes Byte des Hauptspeichers besitzt eine eindeutige Adresse.
b.    Hauptspeicher besteht aus einer sehr langen Liste von Bytes.
c.    Jede Speicherstelle kann eine beliebige Anzahl von Bits speichern.
d.    Speicheradressen sind positive ganze Zahlen und beginnen mit 0.

7. Was sind die zwei grundsätzlichen Dinge, die ein Prozessor mit einem Byte des Hauptspeichers tun kann?

a.    Er kann 8 Bits in ein Speicherbyte schreiben und er kann 8 Bits aus einem Speicherbyte lesen.
b.    Er kann ein Byte des Hauptspeichers löschen und er kann ein Byte des Hauptspeichers lesen.
c.    Er kann Werte aus dem Hauptspeicher einer Summe hinzufügen und er kann den Wert auf den Bildschirm ausgeben.
d.    Er kann ein Byte in den Sekundärspeicher schreiben und er kann ein Byte aus dem Sekundärspeicher lesen.

8. Was ist eine Datei?

a.    Eine Datei ist ein Abschnitt des Hauptspeichers, der für die Speicherung von Daten verwendet wird.
b.    Eine Datei ist eine Sammlung von Information, die einen Namen bekommen hat und die auf dem Sekundärspeicher gespeichert ist.
c.    Eine Datei ist ein Teil eines Programms, der verwendet wird, um zu beschreiben was das Programm tun soll.
d.    Datei ist ein anderer Name für Diskette.

9. Wie werden Bytes auf die Festplatte geschrieben?

a.    Winzige Löcher in der Platte repräsentieren die Bits.
b.    Die Platte ist mit Wachs überzogen, auf dem die Bits als winzige Aufdrücke aufgezeichnet werden.
c.    Bits werden mit Tintenstrahl auf die Platte gedruckt.
d.    Bits werden magnetisch auf der Platte aufgezeichnet.

10. Welche Rolle spielt das Betriebssystem bei der Handhabung von Dateien?

a.    Das Betriebssystem hat mit Dateien nichts zu tun; es hält nur die Hardware am Laufen.
b.    Das Betriebssystem ist nur mit den Dateien befasst, die es selbst braucht.
c.    Alles was im Computersystem mit Dateien gemacht wird bezieht das Betriebssystem mit ein.
d.    Das Betriebssystem greift auf Dateien zu, die Applikationsprogramme enthalten und startet die Applikation, aber die Applikation managt ihre eigenen Dateien völlig allein.
