Bytes, nicht Bits

Die vorherige Tabelle hat die Anzahl von Bytes, nicht Bits aufgelistet. Also entspricht 1 K Speicher 1024 Byte, oder 1024*8 == 8.192 Bit. Man ist üblicherweise nicht besonders an der exakten Anzahl der Bits interessiert.

Es wird für Ihre berufliche Laufbahn sehr wertvoll sein, wenn Sie sicher wissen wie Zweier-Potenzen zu multiplizieren sind. 


2M  *  2N  =  2(M+N)

In dem oben stehenden bedeutet * Multiplikation. Zum Beispiel: 


26  *  210  =  216
