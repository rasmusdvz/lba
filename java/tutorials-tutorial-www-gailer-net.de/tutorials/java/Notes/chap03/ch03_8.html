Inhalte des Hauptspeichers

 

Hauptspeicher (wie jeder Computerspeicher) speichert Bits als Muster. Das heißt, jede Speicherstelle besteht aus acht Bit und jedes Bit ist entweder "0" oder "1". Zum Beispiel zeigt die Abbildung die ersten paar Speicherbytes.

Das Einzige, was an einer Speicherstelle gespeichert werden kann, sind acht Bit, jedes mit einem Wert von "0" oder "1". Die Bits an einer Speicherstelle werden die Inhalte dieser Stelle genannt.

Manchmal werden Leute sagen, dass jede Speicherstelle eine binäre 8-Bit Zahl enthält. Das ist OK, so lange Sie daran denken, dass die "Zahl" verwendet werden könnte, um ein Zeichen darzustellen oder irgendetwas anderes.

Denken Sie daran, dass es vom Zusammenhang abhängt, was ein bestimmtes Muster darstellt (das heißt, wie ein Programm es verwendet). Sie können kein beliebiges Bitmuster betrachten (wie jenes in der Abbildung) und sagen was es darstellt.
