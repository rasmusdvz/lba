Merkmale des Hauptspeichers

Der Hauptspeicher ist für ein Computersystem genauso entscheidend wie der Prozessorchip. Schnelle Computersysteme besitzen sowohl einen schnellen Prozessor als auch einen großen, schnellen Speicher. Hier ist eine Liste von einigen Merkmalen des Computerspeichers. Manche Merkmale sind für beide Arten von Speicher richtig; andere sind nur für eine richtig.
