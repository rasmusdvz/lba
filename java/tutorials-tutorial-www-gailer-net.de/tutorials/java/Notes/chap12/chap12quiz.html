1. Wie viele Alternativen gibt es bei einer einzelnen if-else-Anweisung?

a.    1
b.    2
c.    3
d.    4

2. Was gibt das folgende Codefragment auf den Bildschirm aus?

int summe = 14;
if ( summe < 20 )
  System.out.print("Unter ");
else
{
  System.out.print("Ueber ");
  System.out.println("dem Limit.");
}
a.    Unter
b.    Ueber
c.    Unter dem Limit.
d.    Ueber dem Limit.
