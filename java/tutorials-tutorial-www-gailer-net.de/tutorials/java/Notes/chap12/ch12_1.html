Kapitel 12 — Entscheidungen treffen

Dieses Kapitel betrachtet wie Computerprogramme Entscheidungen treffen, indem sie die if-Anweisung verwenden. Diese Anweisung ist eine der elementaren Bausteine der Programmierung.

Themen:

  • Zwei-Wege-Entscheidungen
  • Die if-Anweisung
  • Anweisungsblöcke
  • Boole'sche Ausdrücke
  • Relationale Operatoren
  • Beispielprogramme

FRAGE 1:

Sie fahren in Ihrem Auto und es beginnt zu regnen. Der Regen fällt auf ihre Windschutzscheibe und verschlechtert die Sicht. Sollten Ihre Scheibenwischer an oder aus sein?
