Eine Referenzvariable deklarieren

Es gibt verschiedene Möglichkeiten, um eine Referenzvariable zu deklarieren: 

Klassenname variablenname;
  • Das deklariert eine Referenzvariable und deklariert die Klasse des Objekts auf die sie später verweisen wird. Kein Objekt wurde erzeugt. 
Klassenname variablenname  =  new Klassname( parameter, parameter, ... ) ;
  • Das deklariert eine Referenzvariable und deklariert die Klasse des Objekts. Aber jetzt wird, zur Laufzeit, ein neues Objekt konstruiert und eine Referenz auf dieses Objekt wird in die Variable gestellt. Manchmal werden Parameter benötigt, wenn ein Objekt erzeugt wird. 
Klassenname variablenameEins, variablennameZwei ;
  • Das deklariert zwei Referenzvariable, beide verweisen möglicherweise auf Objekte derselben Klasse. Wenn Sie möchten, können Sie das mit mehr als zwei Variablen tun. 
Klassenname variablenameEins   =  new Klassenname( parameter, parameter, ... ),
            variablennameZwei  =  new Klassenname( parameter, parameter, ... ) ;
  • Das deklariert zwei Referenzvariable. Zur Laufzeit werden zwei Objekt erzeugt und ihre Referenzen werden den zwei Variablen zugewiesen. Noch einmal, Sie können das mit mehr als zwei Variablen tun solange Sie dem Muster folgen.
