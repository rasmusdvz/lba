Andere Methoden von String

Es gibt viele String-Methoden. Die folgende Liste enthält nur ein paar. Die komplette Liste und detaillierte Beschreibungen finden Sie in der Java Dokumentation im Internet. 

public char charAt( int index )
public String concat( String str )
public boolean endsWith( String suffix )

public boolean equals( Object anObject )
public boolean equalsIgnoreCase( String anotherString )

public int indexOf( int ch )
public int indexOf( String str )

public int length()
public boolean startsWith( String prefix )

public String substring( int beginIndex )
public String substring( int beginIndex, int endIndex )
public String toLowerCase()

public String toUpperCase()
public String trim()

Einige dieser Methoden werden in den Aufgaben und späteren Kapiteln beschrieben. Für jetzt merken Sie sich nur, dass einige dieser Methoden Daten über den String zurückgeben (so wie length()) und andere neue Strings, auf der Basis des Originalstrings, erzeugen (so wie toLowerCase()).
