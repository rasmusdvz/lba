Alle Klassen eines Pakets importieren

Oft benötigt ein Programm mehr als eine Klasse aus einem bestimmten Paket, wie java.util. In diesem Fall ist es zweckmäßig alle Klassen des Pakets auf einmal zu importieren, wie es das folgende Programm macht: 

import java.util.*;

class ImportDemo04
{
  public static void main ( String[] args )
  {
    Scanner scan ;    // eine Klasse in java.util
    Random rand ;     // eine weitere Klasse in java.util

    scan = new Scanner( System.in );
    rand = new Random( );

    // mache etwas mit dem Scanner- und Random-Objekt (siehe folgende Kapitel)
  }
}

Der Teil import java.util.* bedeutet, importiere alle Klassen des Pakets. Es schadet nicht das zu tun, auch wenn Sie nur eine oder zwei davon brauchen.
