Kapitel 61 Programmieraufgaben

Aufgabe 1 — Temperatur-Konverter

Modifizieren Sie das Temperatur-Converter Programm, so dass es Celsius in Fahrenheit umrechnet. Verwenden Sie die Formel: 

fahr = celsius*9/5 + 32

Zurück zum Inhaltsverzeichnis.

Aufgabe 2 — Zwei-Wege Konverter

Fügen Sie dem Temperatur-Konverter Programm zwei Buttons hinzu. Das Klicken auf den einen Button konvertiert Fahrenheit in Celsius. Das Klicken auf den anderen Button konvertiert Celsius in Fahrenheit. Fügen Sie einen "Reset"-Button hinzu, der den Inhalt der beiden Textfelder löscht.

Zurück zum Inhaltsverzeichnis.

Aufgabe 3 — Anwenderfreundlicher Fett-Kalkulator

Modifizieren Sie das Feld für die Kalorien, so dass es keinen Fehler verursacht, wenn der Anwender Text eingibt, der nicht in ein double konvertiert werden kann. Betrachten Sie dazu das modifizierte Temperatur-Konverter Programm und versuchen Sie etwas Ähnliches.

Zurück zum Inhaltsverzeichnis.

Aufgabe 4 — Anwenderfreundlicher Fett-Kalkulator, mit Hinweis

Modifizieren Sie das Fett-Kalkulator, so dass es ein weiteres Textfeld enthält. Beträgt der Prozentsatz der Fettkalorien 30 und mehr, gibt dieses Textfeld "Zu viele Fettkalorien!" aus. Beträgt der Prozentsatz der Fettkalorien weniger als 30, dann gibt dieses Textfeld "Fettanteil OK." aus.
