Der vollständige Konstruktor

Hier ist das vollständige Programm. Der Konstruktor in der Definition von HalloObjekt hat einen Parameter mit dem Namen str. Dieser Parameter wird verwendet, um Daten an den Konstruktor zu übergeben. Der Konstruktor selbst macht nichts anderes als sie in die Variable mitteilung des Objekts zu kopieren. 

class HalloObjekt
{
  String mitteilung;

  HalloObjekt( String str )
  {
    mitteilung = str;
  }

  void sprich()
  {
    System.out.println( mitteilung );
  }
}

class HalloTester
{
  public static void main ( String[] args )
  {
    HalloObjekt einObjekt = new HalloObjekt("Eine Mitteilung!");
    einObjekt.sprich();
  }
}
