Kapitel 30 — Eigene Klassen definieren

Das vorhergehende Kapitel hat den Einsatz von Objekten und ihrer Methoden besprochen. Die Objekte wurden unter Verwendung der Klassenbibliotheken von Java erstellt. Dieses Kapitel bespricht wie Sie Ihre eigenen Klassen definieren und daraus Ihre eigenen Objekte erstellen können.

Themen:

  • Objektorientierte Programmierung
  • Syntax von Klassendefinitionen
  • Syntax von Methodendefinitionen
  • Die return-Anweisung
  • Der Rückgabewert void
  • Ein objektorientiertes Beispiel "Hello World"
  • Klassen testen
