Mehr Objekte zur Laufzeit erzeugen

In der objektorientierten Programmierung ist es üblich zur Laufzeit weitere Objekte zu erzeugen. Die neuen Objekte könnten zusätzliche Daten handhaben, die das Programm zum Bearbeiten braucht. Die Abbildung zeigt Ihr Geschäft, nachdem Sie ein paar mehr Leute eingestellt haben.

Beachten Sie, dass die Objekte (Mitarbeiter) miteinander kommunizieren können, nicht nur mit  main(), dem Boss. Manche Objekte kommunizieren gar nicht mit  main(), sondern kommunizieren nur mit anderen Objekten des Systems.
