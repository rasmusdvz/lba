Einen Konstruktor verwenden

Hier ist die Klasse einschließlich einer Stringreferenzvariablen. Natürlich können Sie einen anderen Namen dafür verwendet haben. 

class HalloObjekt
{
  String mitteilung;

  void sprich()
  {
    System.out.println( mitteilung );
  }
}

Die Klasse ist noch nicht vollständig, da es noch keinen Weg gibt mitteilung zu initialisieren (wir kommen gleich dazu). Die Klasse wird in einer Testklasse verwendet: 

class HalloTester
{
  public static void main ( String[] args )
  {
    HalloObjekt einObjekt = new HalloObjekt("Eine Mitteilung!");
    einObjekt.sprich();
  }
}
