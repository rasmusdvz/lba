Datenschutz

Der Schutz Ihrer persönlichen Daten wird von gailer-net.de sehr ernst genommen.

Es werden nur die Daten gesammelt, die nötig sind, um das Angebot für Sie sinnvoll zu gestalten. Diese Seite gibt Ihnen Antworten auf die häufigsten Fragen, die zum Thema Datenschutz gestellt werden.

  • Welche Daten werden erfasst?
  • Bestellformular: Es werden nur die Daten erfasst, die für die Auftragsbestätigung und Rechnungserstellung benötigt werden.
  • Wofür werden die Daten verwendet?
  • Bestellformular: Die erfassten Daten werden ausschließlich für die Auftragsbestätigung und die Rechnung verwendet.

Es erfolgt keine Weitergabe an Dritte!

Ohne Ihr ausdrückliches Einverständnis werden von gailer-net.de keine personenbezogenen Daten an Dritte weitergegeben. Allgemeine Daten, die in keiner Verbindung zu personenbezogenen Daten stehen, werden statistisch ausgewertet.
