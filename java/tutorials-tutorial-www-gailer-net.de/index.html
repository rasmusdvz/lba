Zum Java Tutorial interaktiv — Programmieren lernen mit Java

Das interaktive Java Tutorial von Bradley Kjell (Central Connecticut State University) ist eine einsteigerfreundliche Einführung in die Informatik, mit dem Schwerpunkt Java. Bei diesem Tutorial handelt es sich um einen vollständigen Javakurs für Anfänger, unter Einsatz des programmierten Lernens.

Die Lektionen enthalten viele interaktive Elemente und ausführliche Codebeispiele. Nach jeder Lektion folgt eine Testfrage. Am Ende eines jeden Kapitels gibt es einen Multiple Choice oder Lückentest, gefolgt von Programmieraufgaben. Derzeit sind ca.

Das Original (englisch) finden Sie bei chortle.ccsu.edu. Eine französische Übersetzung ausgewählter Kapitel gibt es bei www.lifl.fr. Das Tutorial steht auch als Download zur Verfügung.

Vortrag: Enhancing Usability by International Students of a Distance Learning Web Site von Bradley Kjell, gehalten auf der ISTAS 2004 (International Symposium On Technology And Society 2004). Vortrag im PDF-Format anzeigen.
